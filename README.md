# prjcap

Консольный трекер задач по проектам для agent-режима в браузере.

## Chromium-плагин (рекомендуется)

Основная и рекомендуемая форма использования — браузерное расширение для Chromium-браузеров (Chrome, Edge, Brave и др.).

### Установка

1. Откройте в браузере `chrome://extensions/`
2. Включите «Режим разработчика» (右上角)
3. Нажмите «Загрузить распакованную» и выберите папку `extension/`

### Возможности

**Popup (кнопка расширения):**
- Выбор активного проекта из списка
- Открытие страницы agent-чата и автоматическая вставка текста следующей задачи
- Копирование текста задачи в буфер обмена
- Создание новых задач прямо из интерфейса плагина
- Просмотр очереди задач проекта
- Управление проектами: добавление, редактирование, удаление
- **Лента уведомлений** — список задач запланированных на сегодня с подсветкой ближайшей
- **Кнопка «Расписание»** — быстрый переход на страницу таймлайна

**Расписание (timeline.html):**
- Вертикальный таймлайн на весь день с 15-минутными слотами
- Карточки задач привязаны к конкретному времени и дате
- Задачи имеют настраиваемую длительность (15 мин — 3 часа, кратную 15 мин)
- Красная линия текущего времени, обновляется автоматически
- **Drag & Drop** — перетаскивание задач между слотами и днями
- Двойной клик на слот для быстрого создания задачи
- **Фильтр по проекту** — показать задачи всех проектов или только одного
- **Цветовая кодировка** — каждый проект имеет свой цвет
- Навигация по датам (стрелки ← →, кнопка «Сегодня», клавиши ← → T)
- Модальное окно для создания и редактирования задач

Каждый проект хранит URL чата с агентом, начальный промпт (instruction prefix), финальный промпт (agent tail) и CSS-селектор поля ввода для автовставки.

## Консольный CLI

Предполагается, что у вас есть venv `act_env_general` для Python-окружения, где уже установлены `whisper` и `torch`.

Пример:
```bash
act_env_general
pip install -e .
prjcap --help
```

## Поток

1. Создайте проект:
```bash
prjcap projects add --name "myproj" --chat-url "https://chat.z.ai/..." --instruction-prefix "..."
```

2. Создайте задачу (текст или голос):
```bash
prjcap tasks create --project "myproj" --text "Сделай ..."
prjcap tasks create --project "myproj" --voice
```

3. Отправьте задачу в agent-чат (CLI покажет сообщение для вставки):
```bash
prjcap tasks send --task-id 1
```

4. Отметьте выполнение вручную:
```bash
prjcap tasks done --task-id 1 --response "..."
```
