# prjcap

Трекер задач для agent-режима — Chromium-расширение для управления проектами, задачами и взаимодействия с AI-агентами через чат.

## Установка

1. Откройте в браузере `chrome://extensions/`
2. Включите «Режим разработчика» (右上角)
3. Нажмите «Загрузить распакованную» и выберите папку `extension/`

## Возможности

### Popup (кнопка расширения)
- Выбор активного проекта из списка
- Открытие страницы agent-чата и автоматическая вставка текста следующей задачи
- Копирование текста задачи в буфер обмена
- Создание новых задач прямо из интерфейса
- Просмотр очереди задач проекта
- Управление проектами: добавление, редактирование, удаление
- Лента уведомлений — список задач на сегодня с подсветкой ближайшей
- Кнопка «Расписание» — быстрый переход на страницу таймлайна
- Импорт / экспорт данных в JSON

### Расписание (timeline.html)
- Вертикальный таймлайн на весь день с 15-минутными слотами
- Карточки задач привязаны к конкретному времени и дате
- Настраиваемая длительность (15 мин — 3 часа)
- Красная линия текущего времени
- Drag & Drop — перетаскивание задач между слотами и днями
- Двойной клик на слот для быстрого создания задачи
- Фильтр по проекту
- Цветовая кодировка проектов
- Навигация по датам (стрелки ← →, кнопка «Сегодня», горячие клавиши)
- Модальное окно для создания и редактирования задач
- Кнопка «✓ Сделано» на карточке задачи
- Кнопки 💬 (открыть чат) и 📋 (копировать промпт) на карточке
- Перетаскиваемая ширина панели «Нераспределённые»
- Управление проектами прямо на странице таймлайна
- Голосовой ввод задач (SaluteSpeech)

### Общие
- Каждая задача строит промпт из: instruction prefix + текст задачи + agent tail
- Автоматическая вставка промпта в поле чата агента
- CSS-селектор поля ввода настраивается для каждого проекта
